AMCSM Manifest (Short Version)

AMCSM is based on the idea that a model can speak music – and that the same semantic mechanisms apply directly to voice, prosody, and emotional expression.  
Music is not an audio format but a semantic language communicated through MIDI‑2.0.

AMCSM treats music as a system of meaning, intention, emotion, and interaction.
A model should not merely generate notes, but express musical meaning, interpret it, and respond in real time.

Core principles:

    Music as language — music is communicated, not generated.

    MIDI‑2.0 as semantics — high‑resolution parameters as carriers of expression.

    Emotion as control space — valence/arousal instead of style presets.

    Real‑time interaction — music as dialogue, not output.

    Co‑regulation — human and model influence each other.

    Property exchange — models explicitly describe their capabilities.

    Transfer to voice — musical semantics = prosodic semantics.

AMCSM is a private research project that investigates these ideas and publishes them as open prior art to support free scientific exploration.
