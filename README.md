# amcsm
AMCSM erforscht ein neues KI‑Musikparadigma: Musik wird nicht generiert, sondern über MIDI‑2.0 und Emotion kommuniziert. Echtzeit, Ausdruck, Interaktion. AMCSM explores a new AI‑music paradigm: not generating sound, but communicating music and emotion via MIDI‑2.0. Real‑time, expressive, interactive.
