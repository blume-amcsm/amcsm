# Contributing to AMCSM

Thank you for your interest in contributing to AMCSM – Advanced Musical Communication & Semantic Mapping.

AMCSM is an evolving research project exploring a new paradigm for AI-driven musical interaction.  
Contributions of all kinds are welcome, including:

- Research insights  
- MIDI 2.0 expertise  
- Real-time system design  
- Documentation improvements  
- Code contributions  
- Testing and feedback  

## How to contribute

1. Open an Issue  
   Use GitHub Issues to propose ideas, report problems, or discuss concepts.

2. Fork the repository  
   Create your own branch for changes.

3. Submit a Pull Request  
   Describe clearly what you changed and why.

## Code style

- Keep code readable and well-documented.  
- Avoid unnecessary complexity.  
- Follow existing patterns where possible.

## Communication

Please keep discussions respectful, constructive, and focused on the project goals.

Thank you for helping shape AMCSM.
